Installation sur Android

1

Ouvrez Chrome

Assurez-vous d'utiliser le navigateur Google Chrome sur votre téléphone Android.

2

Appuyez sur le menu

Cliquez sur les 3 points verticaux ⋮ en haut à droite de l'écran.

3

Sélectionnez "Installer l'application"

Dans le menu déroulant, recherchez et appuyez sur "Installer l'application" ou "Ajouter à l'écran d'accueil".

4

Confirmez l'installation

Appuyez sur "Installer" dans la fenêtre qui s'affiche. L'icône Navette Express apparaîtra sur votre écran d'accueil !

Installation sur iPhone / iPad

1

Ouvrez Safari

Cette installation ne fonctionne qu'avec Safari. Ouvrez cette page dans Safari si ce n'est pas déjà fait.

2

Appuyez sur le bouton Partager

En bas de l'écran (iPhone) ou en haut (iPad), appuyez sur l'icône de partage :

3

Faites défiler et appuyez sur "Sur l'écran d'accueil"

Dans le menu qui apparaît, faites défiler vers le bas et sélectionnez "Sur l'écran d'accueil" (icône +).

4

Confirmez en appuyant sur "Ajouter"

Vous pouvez personnaliser le nom si vous le souhaitez, puis appuyez sur "Ajouter" en haut à droite.

L'icône Navette Express apparaîtra sur votre écran d'accueil comme une application native.
